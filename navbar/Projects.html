1 2/4/22 Working with messy data
When collecting data, think carefully how you survey constituents. For example, when seeking someone's job title;
If entered by text box, there will likely be many different responses for the same job (ie. 'software engineer', 'programmer', 'developer', 'SE1')
If entered by a combo box however, with carefully predefined choices, you eliminate this problem (ie. if the only relevant choice is 'Software Developer' then that's what software developers choose)

See this survey for an example of textbox verses combo box data entry.


This project uses data that was collected by a the equivalent of a text box. Much of your work will be to process this data into a useful form.
